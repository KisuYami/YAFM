* Yet Another File Manager
Like ranger, but worse, and in C.

** Install
#+BEGIN_SRC sh
make
sudo make install
#+END_SRC

** Keybinds
|---------+----------------------|
| Key     | Action               |
|---------+----------------------|
| =k=     | Up                   |
| =j=     | Down                 |
| =h=     | Left                 |
| =l=     | Right                |
| =z=     | Hidde/Show dot files |
| =space= | Mark Files           |
| =y=     | Yank                 |
| =d=     | cut                  |
| =D=     | delete               |
|---------+----------------------|
